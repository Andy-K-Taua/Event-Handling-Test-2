# Game Two

Simple Flappy Bird style game using HTML Canvas + vanilla JavaScript.


## Controls

- **Red button** or **hold Space**: Move down  
- **Blue button** or **tap Space**: Move up  
- **Restart button**: Reset game

### Run locally

Just open `index.html` in your browser. No build step needed.

## Project Structure


```text
Event-Handling-Test-2/
├── index.html # Main HTML file
├── main.js # Game logic
├── style.css # Styles

```

---

### How it works
The game spawns obstacles at set intervals with random heights and gaps. Your red square is affected by gravity. Don't crash into the green pipes or the floor/ceiling.
